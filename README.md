# Installation

You may want to test Dry-run by commenting out the command lines.


## Linux, Mac

```sh
git clone https://github.com/ymat2/dotfiles.git
cd dotfiles
sh install.sh  # sudo may be needed
```

## Windows

- Requirements: Git for Windows
- You may have to open PowerShell as administrator.

```sh
git clone https://github.com/ymat2/dotfiles.git
cd dotfiles
./install.ps1
```
